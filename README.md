# 성경구절 PPT 제작 프로그램

개역개정 성경구절 피피티를 제작할 수 있습니다.

## 🎯 주요 기능

### ✅ 현재 구현된 기능

1. **통합된 슬라이드 생성**
   - 하나의 스마트 버튼으로 모든 생성 모드 지원
   - 입력 모드에 따라 자동으로 적절한 생성 방식 선택
   - 단일 구절, 구간 구절, 장 전체 생성 모드

2. **자동 성경 구절 생성**
   - 개역개정 성경 전권 데이터베이스 (31,089 구절)
   - 성경책, 장, 절 선택만으로 자동 내용 생성
   - 실시간 구절 검색 및 표시

3. **PPT 스타일 커스터마이징**
   - 배경색 선택 기능
   - 배경 이미지 업로드 기능 (최대 5MB)
   - 글씨체 색상 선택 기능
   - 실시간 색상 미리보기
   - 맑은 고딕 폰트 볼드체
   - 지능형 자동 줄바꿈 (읽기 편한 구절 분할)

4. **완전한 텍스트 레이아웃 제어**
   - 텍스트 영역 너비 슬라이더 (30%~90%)
   - 텍스트 위치 선택 (좌측/중앙/우측/하단)
   - **하단 위치 시 세로 위치 슬라이더** (20%~80%)
   - 텍스트 크기 조절 (참조: 12-36pt, 구절: 16-48pt)
   - **확장된 줄간격 조절** (0%~200%)
   - 실시간 값 표시 및 즉시 반영

5. **프레젠테이션 모드**
   - 전체화면 슬라이드 쇼
   - 키보드 네비게이션 (화살표, 스페이스바, ESC)
   - 사용자 선택 색상 실시간 적용

6. **PPT 파일 다운로드**
   - PptxGenJS 라이브러리 활용
   - 사용자 선택 색상 적용
   - 자동 파일명 생성 (날짜/시간 포함)

9. **편리한 사용자 인터페이스**
   - **신뢰의 다크 테마** 디자인 (#1B1B1B 배경)
   - **사이버네틱 색상 조합** (보라/시안/민트)
   - 글로우 효과와 애니메이션이 있는 **미래적 UI**
   - 반응형 웹 디자인 (모바일/태블릿 지원)
   - 슬라이드 편집/삭제 기능
   - 템플릿 저장/불러오기 기능
   - 로컬 스토리지 자동 저장

### 🔧 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **라이브러리**: 
  - PptxGenJS (PPT 파일 생성)
  - Font Awesome (아이콘)
  - Google Fonts (나눔고딕)
- **데이터**: 완전한 개역개정 성경 JSON 데이터베이스

### 🎨 주요 기능 세부사항

#### 1. 통합 생성 버튼
- **단일 구절 모드**: 특정 구절 하나만 생성
- **구간 구절 모드**: 시작절~끝절 범위 자동 생성
- **장 전체 모드**: 선택한 장의 모든 구절 자동 생성

#### 2. 색상 커스터마이징
- **배경색**: 색상 선택기로 PPT 배경색 설정
- **텍스트색**: 구절 텍스트 및 참조 색상 설정
- **실시간 미리보기**: 선택한 색상을 즉시 확인 가능

#### 3. 키보드 단축키
- **Ctrl + Enter**: 현재 모드로 슬라이드 생성
- **화살표 키**: 프레젠테이션 네비게이션
- **스페이스바**: 다음 슬라이드
- **ESC**: 프레젠테이션 종료

### 📊 데이터베이스 정보

- **총 구절 수**: 31,089개
- **구약**: 39권 (창세기~말라기)
- **신약**: 27권 (마태복음~요한계시록)
- **데이터 형식**: `{"창1:1": "태초에 하나님이 천지를 창조하시니라", ...}`

### 🚀 사용 방법

1. **성경책 선택**: 드롭다운에서 원하는 성경책 선택
2. **장 번호 입력**: 해당 성경책의 장 번호 입력
3. **생성 모드 선택**: 
   - 단일 구절: 절 번호 추가 입력
   - 구간 구절: 시작절~끝절 입력
   - 장 전체: 장 번호만 입력
4. **색상 및 레이아웃 설정**: 
   - 배경색과 글씨색 선택
   - 텍스트 너비와 위치 조정 (하단 위치 시 세로 위치도 조정)
   - 글씨 크기와 줄간격 조정
5. **슬라이드 생성**: 통합 생성 버튼 클릭
6. **프레젠테이션**: 생성된 슬라이드로 발표 또는 PPT 다운로드

### 💾 저장 기능

- **자동 저장**: 생성된 슬라이드 자동 로컬 저장
- **세션 유지**: 브라우저 재시작 시에도 데이터 유지
- **PPT 다운로드**: .pptx 형식으로 로컬 다운로드

### 🎯 최적화 기능

- **데이터베이스 캐싱**: 성경 데이터 메모리 캐시로 빠른 접근
- **비동기 처리**: 대용량 장 생성 시 UI 블로킹 방지
- **에러 핸들링**: 구절 없음, 네트워크 오류 등 예외 상황 대응

### 📱 반응형 디자인

- **데스크톱**: 최적화된 전체 기능
- **태블릿**: 터치 친화적 인터페이스
- **모바일**: 간소화된 컨트롤과 레이아웃

## 🔄 최근 업데이트 (2024-10-13)

### ✨ 새로운 기능 v2.9 (안정성 개선)
1. **최적화된 폰트 선택 시스템**
   - **5가지 핵심 한글 폰트** 지원 (안정성 우선)
   - 나눔고딕, 노토 산스, 맑은 고딕, 굴림, 돋움
   - **3단계 글씨 굵기**: 보통/볼드/엑스트라볼드
   - 빠른 로딩과 안정적인 렌더링

2. **신뢰와 미래적인 새로운 테마**
   - **다크 그레이 배경** (#1B1B1B)과 **사이버네틱 색상 조합**
   - **보라색** (#5A00E0), **시안** (#00E5FF), **민트** (#A2FFEA) 조합
   - 신뢰감과 전문성을 전달하는 **하이테크 디자인**
   - 글로우 효과와 애니메이션으로 **미래적인 사용자 경험**

3. **개선된 사용자 인터페이스**
   - **슬라이드 생성/관리 버튼을 PPT 스타일 설정 위쪽으로 이동**
   - 더 직관적인 워크플로우 (생성 → 확인 → 스타일링 순서)
   - **PPT 스타일 설정 섹션 새로운 블루 테마** 적용
   - 그라디언트 배경과 애니메이션 효과로 시각적 구분 강화

4. **완전한 텍스트 위치 제어**
   - 텍스트 영역 너비 슬라이더 (30%~90% 조정 가능)
   - 좌측, 중앙, 우측, **하단** 위치 선택
   - **하단 선택 시 세로 위치 슬라이더** (20%~80% 조정 가능)
   - 세밀한 레이아웃 커스터마이징 가능

5. **향상된 줄간격 제어**
   - **줄간격 범위 확장**: 0%~200% (기존 100%~200%)
   - 더욱 세밀한 텍스트 레이아웃 조정 가능
   - 0% 설정으로 매우 촘촘한 텍스트 배치 가능
   - 실시간 값 표시 및 미리보기

6. **최적화된 기본값 설정**
   - 텍스트 크기: **32pt** (구절), **24pt** (참조)
   - 줄간격: **100%** (자연스러운 읽기)
   - 텍스트 영역: **90%** (최대 가독성)
   - 실시간 값 표시 및 미리보기

7. **지능형 PPT 레이아웃**
   - 위치별 자동 Y축 좌표 계산
   - 참조 표시 시 구절 위치 자동 조정
   - 모든 슬라이더 설정이 PPT에 정확히 반영

8. **향상된 템플릿 시스템**
   - 모든 새로운 설정값 자동 저장/복원
   - 세로 위치 설정 포함한 완전한 템플릿 관리
